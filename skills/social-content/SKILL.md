Expert social media strategy with platform-specific guidance, proven templates, and a system for reverse-engineering what already works in your niche.

## Workflow

### Step 1: Understand Context

Gather what you need — ask only what's missing:

| What to know | Why |
|---|---|
| Platform(s) | Each has different formats and audiences |
| Goal | Awareness / leads / traffic / community |
| Audience | Who you're reaching |
| Brand voice | Tone, topics to avoid |
| Existing content | Blog posts, videos, podcasts to repurpose |

### Step 2: Identify the Mode

**CREATE** → Write new posts from scratch or from provided content
**STRATEGY** → Build content pillars, calendar, repurposing system
**OPTIMIZE** → Improve existing posts, analyze performance, fix reach/engagement
**REVERSE ENGINEER** → Find what works in the niche, extract patterns, build playbook

### Step 3: Execute

#### For CREATE mode
1. Read `references/templates.md` for the right platform template
2. Choose a hook formula that fits the content
3. Write the post — apply voice principles (specific > vague, short sentences, write from emotion)
4. Add CTA appropriate for the platform
5. Show the post + offer variations or platform adaptations

#### For STRATEGY mode
1. Read `references/platforms.md` for platform selection guidance
2. Read `references/templates.md` for content pillars framework
3. Define 3–5 content pillars with % mix
4. Build a weekly calendar template
5. Set up repurposing workflow from pillar content

#### For OPTIMIZE mode
1. Ask for current metrics (engagement rate, reach trend)
2. Read `references/analytics.md` for optimization actions
3. Diagnose the problem (low engagement vs. low reach vs. low conversion)
4. Suggest specific changes: hook, format, timing, CTA

#### For REVERSE ENGINEER mode
1. Read `references/reverse-engineering.md` for the 6-step framework
2. Guide user through: Niche ID → Scrape → Analyze → Playbook → Layer Voice → Convert
3. Optionally trigger `apify-lead-generation` skill for scraping competitor posts at scale

## Voice Principles (always apply)

**Specific > Vague**
```
❌ "I made good revenue"    ✅ "I made €47,329"
❌ "It took a while"        ✅ "It took 47 days"
```

**Short. Breathe. Land.**
One idea per sentence. Line breaks liberally. Important points stand alone.

**Write from emotion**
Start with how you felt, not what you did. Show vulnerability when authentic.

**Smart friend tone**
Write like texting advice to a friend. "I found that..." not "You should..."

## Platform Quick Reference

| Platform | Best for | Posting freq | Key format |
|---|---|---|---|
| LinkedIn | B2B, thought leadership | 3–5x/week | Story posts, carousels |
| Twitter/X | Tech, real-time, community | 3–10x/day | Threads, hot takes |
| Instagram | Visual, lifestyle, e-commerce | 1–2x/day | Reels, carousels |
| TikTok | Brand awareness, <35 audience | 1–4x/day | Native vertical video |
| Facebook | Community, local, 35+ | 1–2x/day | Groups, native video |

→ For full platform guides including algorithm tips and what doesn't work: read `references/platforms.md`

## Output Format

After creating content, always show:
1. The post (ready to copy-paste)
2. Where the link goes (if applicable — comments on LinkedIn, bio on Instagram)
3. Best time to post
4. Optional: 1 variation with different hook

For strategy outputs: present as a concise table or calendar the user can act on immediately.
